
Def.
- Let $n\in \mathbb{Z}$, $a\in \mathbb{Z}$, $r\in\{0,\,...,\,n-1\}$
- $\implies$ T.F.A.E
	- $a=r+kn$  for some  $k\in\mathbb{Z}$
	- $a\equiv r\;({\rm mod}\;n)$   { $a$ is congruent to "$r$ modulo $n$" }
	- $r=(a\;{\rm mod}\; n)$   { $r$ is the residue of "$a$ modulo $n$" }
	- $n\;|\;a-r$        { $n$ divides $a-r$ , with no remainder }
