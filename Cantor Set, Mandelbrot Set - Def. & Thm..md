
Def. Mandelbrot Set
- $M\coloneqq\{\,c \in \mathbb{C}\;|\;(P_{c}{}^{\circ^{n}}(0))_{n \in \mathbb{N}}\,$ bounded $\}\subseteq\mathbb{C}$  where  $P_{c}(z)\coloneqq z^{2}+c$  $\forall\,z \in \mathbb{C}$
