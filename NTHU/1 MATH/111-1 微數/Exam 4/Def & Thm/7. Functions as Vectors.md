Function <u>is</u> Vector

Polynomial Function Space <u>is</u> Vector Space
- $\infty$-dimentional
- basis vectors: $1$, $x$, $x^2$, $x^3$...
- e.g.: $3x^3+2x^2+x+1\;\equiv\;[\,1,\,1,\,2,\,3,\,0...\,]$ 
===
- differential is $\infty$-dimentional matrix
