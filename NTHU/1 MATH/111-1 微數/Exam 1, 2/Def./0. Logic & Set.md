Word Usage

(2)

- Lemma:       引理 pre 定理
- Corollary:   系理 post 定理

(.)

- Remark / Note

Logic Notation

- exists: $\exists$, unique: $!$
- and: $\land$, or: $\lor$, negation: $\neg$
