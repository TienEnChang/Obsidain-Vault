
|          | Sensory    | Short-term   | Long-term                 |
| -------- | ---------- | ------------ | ------------------------- |
| Last     | 0 ~ 1s     | 15 ~ 25s     |                           |
| Form     | exact copy | condensed copy | vivid imagery (emotional) |
| Capacity |            | limited      | unlimited                 |

Memory Types
- Sensory
	- Iconic = Visual
	- Echoic = Audio
- Long-termed
	- Procedural  (肌肉記憶): habit, skill
	- Declarative (抽象記憶)
		- Semantic    (知識): knowledge
		- Episodic    (經歷): experience
