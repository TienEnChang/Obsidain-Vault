Def.
- total expenditure ($TE$) / revenue ($TR$):  $Q^d(\bar{p})\cdot \bar{p}$
- total utility ($TU$):      $[\displaystyle\int P^d(q)dq\,]\biggm|_{0}^{Q^d(\bar {p})}$
- consumer surplus ($CS$):   $[\displaystyle\int Q^d(p)dp\,]\biggm|_{\bar {p}}^{P^d(0)}$  i.e. $\,TU-TE$  

Prop.
- $\Delta CS:=CS(\bar {p}_2)-CS(\bar {p}_1)=[\,-\!\!\displaystyle\int Q^d(p)dp\,]\biggm|_{\bar {p}_1}^{\bar {p}_2}$

Rmk.
- $\bar{E}_1:=E^h(\bar{p}_{old},\,\bar{U}_{old})=E^h(\bar{p}_{new},\,\bar{U}_{new})$
- $\bar{E}_2:=E^h(\bar{p}_{new},\,\bar{U}_{old})$                  $\bar{E}_3:=E^h(\bar{p}_{old},\,\bar{U}_{new})$

Def.
- total price effect ($PE$):  $x^m(\bar{p}_{new},\,\bar{E}_1)-x^m(\bar{p}_{old},\,\bar{E}_1)$
	- meaning:
		- when  $\bar{p}_{old}\to\bar{p}_{new}$  causes  $\bar{U}_{old}\to\bar{U}_{new}$
		- what is $\Delta x$ ?                          { $\Delta \bar{E}=0$ }
	
- substitution effect ($SE$):  $x^h(\bar{p}_{new},\,\bar{U}_{old})-x^h(\bar{p}_{old},\,\bar{U}_{old})$
	- meaning:
		- what is  $\Delta x$  s.t.  $\bar{p}_{old}\to\bar{p}_{new}$  with  $\bar{U}_{old}$ ?  { $\Delta \bar{E}=\bar{E}_2-\bar{E}_1$  }
	
- income effect ($IE$):   $x^h(\bar{p}_{new},\,\bar{U}_{new})-x^h(\bar{p}_{new},\,\bar{U}_{old})$
	- meaning:
	    - what is  $\Delta x$  s.t.  $\bar{U}_{old}\to\bar{U}_{new}$  with  $\bar{p}_{new}$ ?  { $\Delta \bar{E}=\bar{E}_1-\bar{E}_2$ }

Def.
- compensating varation ($CV$):  $E^h(\bar{p}_{new},\,\bar{U}_{old})-E^h(\bar{p}_{old},\,\bar{U}_{old})$
	- meaning:
		- when  $\bar{p}_{old}\to\bar{p}_{new}$  causes  "$\bar{U}_{old}\to\bar{U}_{new}$"
		- what is  $|\Delta \bar{E}|$  s.t.  "$\bar{U}_{new}\to\bar{U}_{old}$"  with  $\bar{p}_{new}$ ?  { on util }
			
		- { $\Delta x=-IE$  at  $\bar{p}_{new}$ }
	
- equivalent variation ($EV$):  $E^h(\bar{p}_{new},\,\bar{U}_{new})-E^h(\bar{p}_{old},\,\bar{U}_{new})$
	- meaning:
		- when  "$\bar{p}_{old}\to\bar{p}_{new}$"  causes  $\bar{U}_{old}\to\bar{U}_{new}$
		- what is  $|\Delta \bar{E}|$  s.t.  "$\bar{p}_{new}\to\bar{p}_{old}$"  with  $\bar{U}_{new}$ ?  { on price }
			
		- { $\Delta x=-SE$  at  $\bar{U}_{new}$ }


Prop.  $PE=SE+IE$

Prop.
- $\bar{E}_1=[E(\bar{x}_1;\;\bar{p}_{old})]^{(1)}=[E(\bar{x}_1+SE+IE;\;\bar{p}_{new})]^{(3)}$
- $\bar{E}_2=[E(\bar{x}_1+SE;\;\bar{p}_{new})]^{(2)}=[E(\bar{x}_1+SE+IE+\Delta x_{CV};\;\bar{p}_{new})]^{(4a)}$
- $\bar{E}_3=[E(\bar{x}_1+SE+IE+\Delta x_{EV};\;\bar{p}_{old})]^{(4b)}$

Prop.
- (1)  $\bar{x}_1=x^m(\bar{p}_{old},\,\bar{E}_1)$
- (2)  $\bar{x}_1+SE=x^m(\bar{p}_{new},\,\bar{E}_2)=\bar{x}_1+SE+IE+\Delta x_{CV}$ (4a)
- (3)  $\bar{x}_1+SE+IE=x^m(\bar{p}_{new},\,\bar{E}_1)$
- (4b) $\bar{x}_1+SE+IE+\Delta x_{EV}=x^m(\bar{p}_{new},\,\bar{E}_3)$

- ${Q^d}_{(h,\,\rm old)}(p)=x^h(p,\,\bar{U}_{old})$,  ${Q^d}_{(h,\,\rm new)}(p)=x^h(p,\,\bar{U}_{new})$

Prop.
- For "normal good":    $|CV|>|\Delta CS|>|EV|$  when  $p$↑
- For "inferior good":  $|CV|<|\Delta CS|<|EV|$  when  $p$↑
- For "$IE\approx 0$":         $|CV|\approx|\Delta CS|\approx|EV|$  when  $p$↑

	- $|CV|=|\Delta CS|=|EV|$   since  ${Q^d}_m(p)={Q^d}_{(h,\,\rm old)}(p)={Q^d}_{(h,\,\rm new)}$
